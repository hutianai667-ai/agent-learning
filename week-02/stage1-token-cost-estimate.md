# Stage 1 - Token / Cost Estimate

## 1. 估算任务

任务：

```text
让模型总结一篇 1000 字中文学习笔记，并输出 300 字总结。
2. 估算规则
为了练习估算方法，今天使用假设价格，不代表真实模型价格。
假设：
输入：每 1000 token = $0.001
输出：每 1000 token = $0.003
粗略换算：
1000 个中文字 ≈ 1000 input tokens
300 个中文字 ≈ 300 output tokens
3. 计算过程
输入成本：
1000 / 1000 * 0.001 = $0.001
输出成本：
300 / 1000 * 0.003 = $0.0009
总成本：
$0.001 + $0.0009 = $0.0019
4. 结论
这次总结任务的假设成本约为：
$0.0019
5. 我的理解
Cost 不只看问题本身，还要看输入 token 和输出 token。
长文件会增加输入 token。
长回答会增加输出 token。
多轮对话和工具返回内容也会增加 token。
真实使用 API 时，需要查看官方最新价格，不能用固定价格乱算。
