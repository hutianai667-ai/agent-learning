# Branch Practice - 2026-06-16

## 今日主题

Stage 0 下半：Git branch 正式练习。

## 我的理解

branch 是从 main 分出来的一条临时路线。

我可以在 branch 上修改文件、commit、push，而不是直接在 main 上操作。

## 今日命令

- `git branch`：查看分支。
- `git switch -c branch-practice-2026-06-16`：创建并切换到新分支。
- `git switch main`：切回 main。
- `git push -u origin branch-practice-2026-06-16`：把新分支推送到 GitHub。
