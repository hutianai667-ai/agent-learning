## 今日阅读

今天理解了 Python 文件读写：

- `open()` 用来打开文件。
- `read()` 用来读取文件内容。
- `write()` 用来写入文件内容。
- `with` 可以让 Python 自动关闭文件。
- `as file` 是给打开后的文件对象起一个临时变量名。
- `"r"` 是读取模式，`"w"` 是写入模式，`"a"` 是追加模式。
