## Key Concepts

- git init turns a folder into a Git repository.
- git status shows the current repository state.
- git add moves changes into the staging area.
- git commit saves a version record locally.
- git push uploads local commits to GitHub.

## Practice

I will practice creating a new file, adding it, committing it, and pushing it to GitHub.


## 今日阅读理解

1. `git status` 用来查看当前 Git 仓库状态，包括有没有新文件、修改、暂存内容，以及本地和 GitHub 是否同步。
2. `git commit` 是把已经 `git add` 的修改保存成一次本地记录。
3. `git push` 是把本地已经 commit 的记录上传到 GitHub。
4. `python week-01\scripts\hello.py` 是运行 `week-01\scripts` 目录下的 `hello.py` Python 脚本。

## 今日一句话总结

今天我理解了 Git 的基本流程：修改文件 -> git status -> git add -> git commit -> git push。
